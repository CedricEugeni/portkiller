# Screenshots

Add your screenshots here:
- `screenshot.png` - Main app interface (used in README)
- `demo.gif` - Animated demo showing the workflow (optional but recommended)

## Taking Screenshots

1. Open PortKiller
2. Make sure some ports are visible
3. Take screenshot: Cmd+Shift+4, then Space, click on the popover
4. Optionally hover over a row to show the Stop button
5. Save as `screenshot.png` in this folder

## Creating a GIF Demo

Use tools like:
- **Kap** (https://getkap.co/) - Free, simple
- **LICEcap** - Lightweight
- **QuickTime** + gifski

Show:
1. Opening the app from menu bar
2. Searching for a port
